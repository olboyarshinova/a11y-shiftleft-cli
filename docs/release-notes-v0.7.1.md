# v0.7.1 Release Notes

`a11y-shiftleft-cli` v0.7.1 is a documentation and visual-report polish release
for the stable `audit` workflow.

## Improved

- Simplified the README for first-time users.
- Refreshed the README screenshots from the current demo visual audit.
- Kept the primary path focused on `audit --url ... --out reports --open`.
- Moved advanced commands and report details into collapsed README sections.
- Made full-page screenshot previews more compact in the visual HTML report.

## Update

```bash
npm install --save-dev a11y-shiftleft-cli@0.7.1
```
